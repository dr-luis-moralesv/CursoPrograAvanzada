# Tipos de datos en Python
Python usa un modelo de tipos dinámicos, donde no las variables se ajustan de forma automática dependiendo del valor asignado. 

Se reconocen los siguientes tipos de datos:
- Texto: **str**
- Números: **int**,**float**,**complex**
- Secuencias: **list**,**tuple**,**range**
- Mapeo (diccionarios): **dict**
- Conjuntos: **set**,**frozenset**
- Boleano: **bool**
- Binario: **bytes**,**bytearray**,**memoryview**
- Ninguno: **NoneType**
