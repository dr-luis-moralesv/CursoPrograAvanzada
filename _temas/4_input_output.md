# Salida a consola

La salida estándar en consola se ejecuta mediante la función `print()`.

```python
print("Hola mundo!")
```

La función `print()` admite comillas dobles o comilla sencilla.

```python
print("Hola mundo!")
print('Hola mundo!')
```
## Impresión de variables

Impresión de variables.

```python
x = 45
print("El valor de la variable x es",x,"!")
```

Impresión usando cadena de formato

```python
x = 45
print(f"El valor de la variable x es {x} !")
```

## Entrada de datos

Leer datos desde la consola en Python es muy sencillo, se usa la función `input()`, dentro del paréntesis se coloca el mensaje deseado y la función retorna el valor convertido al tipo de dato identificado.

```python
num = input("Escribe un número: ")
print("El número es ",num)
```
