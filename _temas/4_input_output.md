# Salida a consola

La salida estándar en consola se ejecuta mediante la función `print()`.

```python
print("Hola mundo!")
```

La función `print()` admite comillas dobles o comilla sencilla.

```python
print("Hola mundo!")
print('Hola mundo!')
```
## Impresión de variables

Impresión de variables.

```python
x = 45
print("El valor de la variable x es",x,"!")
```

Impresión usando cadena de formato

```python
x = 45
print(f"El valor de la variable x es {x} !")
```

## Entrada de datos

```python
num = input("Escribe un número: ")
print("El número es ",num)
```
